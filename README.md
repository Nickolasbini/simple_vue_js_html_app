# simple_vue_js_html_app

Aplicação de Lista de Tarefas construída com **Vue.js 3** e **Tailwind CSS** via CDN — sem instalar nada, sem build, sem configuração.

Projeto desenvolvido no vídeo **"Como criar sua primeira aplicação Vue.js do zero"** do canal [Código Prático](https://www.youtube.com/@CodigoPraticoOficial).

📺 **Assista ao vídeo completo:** [LINK_DO_VIDEO]

---

## O que você vai aprender com esse projeto

- Como incluir o Vue.js 3 via CDN em qualquer HTML
- Reatividade com `ref()` — a base de tudo no Vue
- Dados calculados automaticamente com `computed()`
- As diretivas essenciais: `v-model`, `v-for`, `v-if`, `v-bind` e `@click`
- Como estruturar o `setup()` com estado, computed e métodos

---

## Funcionalidades

- Adicionar tarefas (botão ou tecla Enter)
- Marcar tarefas como concluídas via checkbox
- Filtrar por: Todas / Pendentes / Feitas
- Contador reativo de total, feitas e pendentes
- Remover tarefa individual (hover no item)
- Limpar todas as tarefas concluídas de uma vez

## Como rodar

Não precisa instalar nada. Apenas clone o repositório e abra o arquivo no navegador:

```bash
git clone https://github.com/Nickolasbini/simple_vue_js_html_app
cd simple_vue_js_html_app
```

Abra o `index.html` diretamente no navegador — ou use o **Live Server** do VS Code para ver as mudanças em tempo real.
